# Optic Counting Tool


### Scope

Build a tool to count optics and Devices models from multiple sources: IB, ROCE and CUTSHEETS. The script should output counts for the end user, including overall total and break downs by DH and individual racks. A front end should be included for ease of use for non technical end users.
